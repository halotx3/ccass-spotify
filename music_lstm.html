AI composes music

What is the goal?

The work done here is utilizing very small part of MIDI. We have samples of MIDI formatted files by famous composers. We extract only Piano part of the MIDI which is used for training our model to learn the style of the composer. After learning process of the composer's style, a model is created. We can create infinate number of the composer's style music with it.

Toolbox

  1. music21 python package, http://web.mit.edu/music21/
  2. Tensorflow
  3. Pandas, Numpy
  4. Google Colab or NVidia GPU equipped Linux machine.

Preprocess - Extraction of training data from MIDI

MIDI consists of cords and notes for each channnel

  • Cords: A cord is a combination of notes.
  • Notes: A note contains information about pitch, octave and offset.

"music21" is a Python lib for extracting notes and cords from MIDI file.

MIDI is meant to coordinate many different instrument at the same time.

We will pick only one-channel (1st channel of Piano) to simplify training & learning process.

Procedure

1. Extraction of training data

Extract Notes and Cords from Channel 1 (Piano) of MIDI data.

2. Conversion data to numeric sequence

Put all notes and cords into sequential list. Convert categorical (e.g. E3, A3, B, ...) into a set of distict numeric (0,1,2,3,...). Create "Vacab_table" stores mapping between categorical and numeric.

3. Training to create model

As input sequence (Training) for the network is fed into the neural network. The output for each input sequence is predicted as a note representation (Forecasting). Keep moving the sliding window and training the model to predict the next note from each training sequences. Run many EPOCHs of model training until errors are not improved any more.

Note: Using i9 8-core vs Nvidia Tesla, Intel i9 8-core is about 40 times slower than Nvidia Tesla's. It tooks 8~12hours to complete model training using Nvidia Tesla.

LSTM (Long Short Term Memory) layer is a common choice for this type of application.

def create_network(network_input, n_vocab):
    """ create the structure of the neural network """
    model = Sequential()
    model.add(Bidirectional(LSTM(512,
        input_shape=(network_input.shape[1], network_input.shape[2]),
        return_sequences=True)))
    model.add(Dropout(0.3))

    model.add(Bidirectional(LSTM(512,return_sequences=True)))
    model.add(Dropout(0.3))

    model.add(Bidirectional(LSTM(512,return_sequences=True)))

    model.add(Dense(256))
    model.add(Dropout(0.3))

    model.add(Flatten())
    model.add(Dense(n_vocab))
    model.add(Activation('softmax'))
    model.compile(loss='categorical_crossentropy', optimizer='adam')

    return model

4. Save model & vocab_table

Save "model" and "vocabtable". "vocabtable" is conversion map between categorical notes and numeric conversion.

5. Create numeric sequence

Feed the first slding window (this can be any notes or even random notes). You need to feed the first sliding window. The model will keep predicting the next note by following the pattern learned from model training.

6. Conversion to MIDI from numerice sequence

Create MIDI file by using a set of predictions. This is the reverse of the first process.
